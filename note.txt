setup magento 248 p3 với docker trên vps chưa cài gì và host mgthemes.info và ip vps 157.20.83.37
mysql  redis opensearch phpmyadmin php-8.4
hướng dẫn cụ thể từng bước và đặt file ở đâu ( bỏ qua cài docker ) 
trong file docker-compose.yml cach tên đặt thêm tiền tố trước để phân biệt chỉ dành cho dự source này
composer create-project --repository-url=https://repo.magento.com/ magento/project-community-edition=2.4.8-p3 magento248p3

setup docker cho 2 môi trường dev và prod dễ dàng chuyển đổi :
dùng docker-compose.override.yml cho môi trường dev
dùng docker-compose.prod.yml cho môi trường prod
trong file yml này chỉ định các biến môi trường cần thiết cho từng môi trường và mount volume phù hợp, tên đặt thêm tiền tố trước để phân biệt chỉ dành cho dự source này
